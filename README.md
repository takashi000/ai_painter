# ai_painter

Diffusionモデルを使用したお絵かきツールです。

## アップデート

2025/10/15：初回リリース(Stable Diffusion 1.5のみ対応)\
2025/10/19：0.0.2リリース(メニューバー整理、undo, redo機能追加)

## インストール

```sh
uv venv -p 3.12 .venv
```

```sh
git clone https://github.com/takashi000/ai_painter.git
```

```sh
uv pip install ai_painter
```
